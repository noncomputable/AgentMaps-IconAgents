Class: Spritegent

Spritegent(lat_lng, icon_options, marker_options, agentmap)

new Spritegent(lat_lng, icon_options, marker_options, agentmap)

Constructor for the Spritegent class, using Leaflet class system.
Parameters:
Name Type Description
lat_lng LatLng A pair of coordinates to place the spritegent at.
icon_options Object An array of options for the Leaflet Icon depicting of the Spritegent.
marker_options Object An array of options for the Leaflet Marker representing the Spritegent.
agentmap Agentmap The agentmap instance in which the spritegent exists.
Properties:
Name Type Attributes Default Description
agentmap Agentmap The agentmap instance in which the spritegent exists.
place Place A place object specifying where the spritegent is currently at.
steps_made number <optional>
 0 The number of steps the spritegent has moved since the beginning.
this.trip Object Properties detailing information about the spritegent's trip that change sometimes, but needs to be accessed by future updates.
Properties
Name Type Attributes Description
moving boolean Whether the spritegent currently moving.
paused boolean Whether the spritegent should be allowed to move along its trip.
current_point Point <nullable>
 The point where the spritegent is currently located.
goal_point Point <nullable>
 The point where the spritegent is traveling to.
lat_dir number <nullable>
 The latitudinal direction. -1 if traveling to lower latitude (down), 1 if traveling to higher latitude (up).
lng_dir number <nullable>
 The longitudinal direction. -1 if traveling to lesser longitude (left), 1 if traveling to greater longitude (right).
speed number <nullable>
 The speed that the spritegent should travel, in meters per tick.
angle number <nullable>
 The angle between the current point and the goal.
slope number <nullable>
 The slope of the line segment formed by the two points between which the spritegent is traveling at this time during its trip.
path Array A sequence of LatLngs; the spritegent will move from one to the next, popping each one off after it arrives until the end of the street; or, until the trip is changed/reset.
controller function <nullable>
 User-defined function to be called on each update (each tick).
fine_controller function <nullable>
 User-defined function to be called before & after each movemnt (on each step an spritegent performs during a tick).
